# Rad's plugin marketplace

## Plugins

### Architecture planning - Rad (`architecture-planning-rad`)

Produces engineering-grade Architecture & Build Plan PDF documents for any software project:
phased work packages with dependency linkage, evidence-demandable acceptance gates, four
vector diagrams (system context, entity model, state machine, dependency graph), and the
complete task-level implementation plan embedded as Appendix A.

## Install

**Claude Code**

```
/plugin marketplace add Zen-CIV/architecture-planning-marketplace
/plugin install architecture-planning-rad@rad-plugins
```

**Cowork (Team/Enterprise)**: Organization settings > Plugins > Add plugin > GitHub >
enter `Zen-CIV/architecture-planning-marketplace`.
